# JavaScript - Final Project

**Objective:** Fix and improve the functions so that all test cases pass.

## 📖 Instructions

1. Install the `jest` package by running:

   ```bash
   npm i -D jest
   ```

   This will add the package under `devDependencies` in the `package.json` file.

2. The exercises are located inside the `src/` directory while the test cases are located inside the `tests/` directory.
3. Test each exercise individually by running:

   ```bash
   npm test -- src/ex01.js
   ```

   You can also test ALL exercises at once:

   ```bash
   npm test -- --verbose
   ```

## 💾 Commits

Make sure to commit after working on an exercise. **DO NOT** commit your changes all at once!
